# UAS_PDK_AI_COLAB
Analisis Sentimen Ulasan Produk E-Commerce Menggunakan Model Transformer (BERT)
Repository ini berisi implementasi analisis sentimen ulasan produk e-commerce menggunakan model BERT untuk mengklasifikasikan ulasan menjadi positif dan negatif. Proyek mencakup preprocessing data teks, fine-tuning model, serta evaluasi performa menggunakan metrik klasifikasi.
